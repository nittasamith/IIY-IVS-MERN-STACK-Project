# 🌐 Time-Based Employment Platform for Rural and Urban Workforce

## 📌 Overview
The **Time-Based Employment Platform** is a full-stack web application built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**.  
It aims to bridge the gap between **job seekers and job providers** by offering flexible, time-based employment opportunities.

This platform focuses on reducing unemployment and improving workforce utilization across both **rural and urban areas**.

---

## 🚀 Features

### 👨‍💼 For Job Seekers
- 🔍 Search jobs based on time (hourly, daily, weekly, monthly)
- 📂 Filter jobs by category, location, and type
- 📝 Apply for jobs easily
- 🔔 Receive notifications for new opportunities

### 🏢 For Job Providers
- ➕ Post time-based job listings
- ✏️ Manage and update job postings
- 📊 View applicants and manage hiring

### 🌟 General Features
- 🔐 User authentication (Login/Register)
- 📱 Responsive UI
- 📊 Transparent wage details
- 🚫 Eliminates middlemen

## ⚙️ Installation & Setup

---

### 2️⃣ Setup Backend

cd backend
npm install
npm start


---

### 3️⃣ Setup Frontend

cd frontend
npm install
npm run dev

## 🔐 Environment Variables
